## 제출방법

1. 숙제 제출 폴더에 자기 이름으로 폴더 만들기
2. 자기가 만든 폴더 안에 html과 css파일 만들어서 작업하기 (파일 이름 상관없음)
3. 완성화면 캡쳐해서 자기 폴더 안에 올리기 
(파일 이름 상관없음)
4. github 올리기 = commit하기
    방법 ->
    1. github Desktop 열기
    2. 26_learn 들어가기
    3. Changes클릭 -> Summary (required)에 원하는 제목 쓰기
    4. Description에 원하는 설명 쓰기
    5. Commit ~~ files to main 파란 버튼 클릭해서 올리기
    6. 위에 Push origin을 클릭하여 올리기 
        -이걸 해야지 올라감 
        (다른 사람것도 보고싶다면 그 위치에 있는 버튼을 누르면 받아짐)

모르겠다면 선예린 또는 최희섭한테 톡하기!
